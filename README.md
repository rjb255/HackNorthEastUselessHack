# HelpfulBot

This is a discord bot written in Node.js that aims to be as inconvenient as possible to the members of the server it is on.

## Table of Contents
* [Features](#Features)
* [Images](#Images)
* [Setup](#Setup)
* [Usage](#Usage)

## Features
* Deletes messages
* Kicks members
* Sent messages are embedded Rick-Rolls
* Replies to 'ping' with 'pong'
